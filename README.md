# memoricon-masi-lite
Memoricon MASI Lite — AI system for capturing, structuring, verifying, and retrieving real-world events as trusted memory.
Memoricon MASI Lite

Memoricon MASI Lite is an AI system that captures real-world events, structures them into verified memory, and allows users to retrieve trusted information later.

1. Source Code

This repository contains all source code required to run the Memoricon MASI Lite prototype, including:

Frontend interface for capturing events

Backend services for processing and storage

AI integration for extraction and reasoning

Memory database implementation

The system demonstrates how real-world observations can be transformed into structured, verifiable memory.

2. Video Introduction

A demonstration video of the project is available at:

👉 (https://youtu.be/R0FRrW91k7E)

The video explains:

(a) Features

Event capture using text or voice

AI extraction of key event elements

Structured reasoning using the MASI framework

Storage of verified memories

Retrieval of relevant memories with citations

(b) How Memory Is Used

The system converts user observations into structured memory entries.

Each memory is processed through:

Event capture

AI extraction of components

Structured representation

Verification

Storage in a searchable memory database

When a user asks a question, the system retrieves the most relevant stored memories and uses them to generate a grounded response.

(c) How This Memory Helps Users

This memory system helps users:

Recall past events accurately

Reduce information loss

Support decision-making

Maintain operational knowledge over time

Access verified information instead of relying on memory alone
